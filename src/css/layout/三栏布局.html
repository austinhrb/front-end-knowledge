三栏布局

1. float布局

注意事项

  • 中间栏必须放在最后面，防止其占据文档流，将右浮动元素挤下去
  • 中间栏需要加左右margin，防止中间栏高度超出内容向左右溢出
  • 注意清除浮动
.left {
  float: left;
  width: 200px;
}
.right {
  float: right;
  width: 200px;
}
.center {
  margin-left: 200px;
  margin-right: 200px;
}

2. 绝对定位布局

.left {
  position: absolute;
  left: 0;
  width: 200px;
}
.center {
  position: absolute;
  left: 200px;
  right: 200px;
}
.right {
  position: absolute;
  right: 0;
  width: 200px;
}

3. flex布局

父元素样式：

.flex-layout {
  display: flex;
}
.left {
  width: 200px;
}
.center {
  flex: 1;
}
.right {
  width: 200px;
}

4. grid布局

父元素样式：

.grid-layout {
  display: grid;
  grid-template-columns: 200px auto 200px;
}

5. table布局

不推荐使用table做布局

父元素样式

.table-layout {
  display: table;
  width: 100%;
}
.table-layout .left {
  display: table-cell;
  width: 200px;
}
.table-layout .center {
  display: table-cell;
}
.table-layout .right {
  display: table-cell;
  width: 200px;
}
